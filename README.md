# DashDish 

Welcome to **DashDish**! A beautiful, feature-rich food delivery Flutter application that makes browsing and ordering your favorite meals seamless and delightful.

##  Demo

Check out DashDish in action!

![DashDish Demo](assets/demo.gif)

##  Features

- **Intuitive UI/UX**: A clean and modern interface for navigating menus and discovering new dishes.
- **Food Details**: Comprehensive views for each food item including imagery, descriptions, and portion selections.
- **Seamless Checkout**: A smooth, robust checkout process for quickly placing and reviewing your orders.
- **Responsive Navigation**: State-of-the-art navigation powered by a centralized router.

##  Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (latest stable version)
- An IDE of your choice (VS Code, Android Studio, etc.)

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/olit2004/dashdish.git
   ```

2. Navigate to the project directory:
   ```sh
   cd dashdish
   ```

3. Install the required Flutter dependencies:
   ```sh
   flutter pub get
   ```

4. Run the app on your preferred emulator or connected device:
   ```sh
   flutter run
   ```

## Project Structure

This project follows a modular, feature-based architecture for scalability and maintainability:

- `lib/features/food/` - Contains food-related screens like the food details page.
- `lib/features/order/` - Manages user order flows, including the checkout page.
- `lib/core/routing/` - Centralized routing logic (`app_router.dart`).
